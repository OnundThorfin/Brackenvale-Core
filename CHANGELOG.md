# Brackenvale Core Changelog

All notable changes to the Brackenvale Core module are documented here.

This project uses milestone-based versioning. Each release represents a stable checkpoint that can be deployed to The Forge.

---

## v0.3.6 — In Progress

### Planned

- Begin rebuilding Character Sheet Page 1 from the canonical Brackenvale PDF.
- Establish the final print-inspired visual language.
- Add the Page 1 header, identity block, and combat panel.
- Keep the sheet rendering stable throughout development.

---

## v0.3.5 — Rendering Foundation

Released: July 21, 2026

### Added

- Rebuilt the character sheet from a minimal foundation.
- Verified custom template rendering.
- Verified custom stylesheet loading.
- Verified custom sheet registration in Foundry VTT 14.
- Established a stable baseline for future sheet development.

### Fixed

- Removed rendering errors caused by premature data preparation.
- Eliminated the Handlebars `toFixed` formatting crash.

---

## v0.3.4 — Initial Character Sheet Architecture

Released: July 21, 2026

### Added

- Initial four-page character sheet architecture.
- Custom character sheet registration.
- Initial Handlebars template and stylesheet.
- Automated deployment through GitHub Actions.

### Notes

This release established the technical framework but was not production ready.
